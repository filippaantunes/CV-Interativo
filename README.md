# CV Interativo & Portfólio Multimédia

Bem-vindo ao meu repositório de percurso profissional e criativo! Este projeto é uma experiência de **UI/UX** desenhada para apresentar a minha trajetória de vida, formação e projetos de forma interativa.

## O Projeto
Este portfólio funde o currículo tradicional com a tecnologia moderna, destacando:
* **Percurso de Vida:** Formação académica e profissional detalhada.
* **Projetos de Negócio:** Experiência estratégica e empreendedora.
* **Mundo Gaming:** Concept Art e propostas de Game Design para o universo Evony.

## Tecnologias Utilizadas
* **HTML5 & CSS3:** Estrutura e estilização personalizada.
* **JavaScript:** Interatividade e janelas modais.
* **AI-Assisted Design:** Criação de ativos visuais de alta fidelidade.

## Destaques de Design
O layout foi concebido para ser imersivo, utilizando uma estética de "Pergaminho Antigo" para a seção de gaming e um design limpo e profissional para as seções de formação.
